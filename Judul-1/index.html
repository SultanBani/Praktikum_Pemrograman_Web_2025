Pengetahuan dan Fokus Utama

Konsep Smart System

Fokus saya adalah pada bagaimana mengintegrasikan perangkat keras (hardware) dengan algoritma cerdas (software) untuk membuat keputusan mandiri. Ini mencakup pemahaman mendalam tentang siklus data hingga pemrosesan (ML).

Keahlian Machine Learning (ML)

Saya cukup bisa dalam menggunakan framework seperti Scikit-learn untuk implementasi algoritma dasar seperti Regresi Linier dan Klasifikasi. Pengetahuan ini menjadi dasar untuk membangun model prediktif pada sistem cerdas.
